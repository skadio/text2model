# Text2Zinc Code
This Repository contains all code associated with the Text2Zinc Paper. The leaderboard can be found here: [Text2Zinc HuggingFace Leaderboard ]([url](https://huggingface.co/spaces/skadio/text2zinc-leaderboard)). The dataset can be found [here]([url](https://huggingface.co/datasets/skadio/text2zinc)).
The following files contain agents that were used for baseline submissions to the dataset:
- generate_code.py
- generate_code_stitch.py
- generate_two_stage_code.py

This repository also contains all prompts used in the agents and the lark grammar for MiniZinc used in structured prediction.
